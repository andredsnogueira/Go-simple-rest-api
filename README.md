# Go Simple REST API

### Instalation & Running

Requires [Go](https://golang.org/) and [Mux](https://github.com/gorilla/mux)

Install the dependencies and start the server.

```sh
$ cd Go-simple-rest-api
$ go get -u github.com/gorilla/mux
$ go run *.go
```
